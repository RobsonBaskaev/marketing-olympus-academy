# Маркетинг: от азов до Олимпа

Первая рабочая версия образовательной платформы по маркетингу. В ней уже есть лендинг, карта обучения, сценарии для специалистов, компаний и университетов, а также интерактивная входная диагностика с персональной рекомендацией.

## Как запустить

1. Установите Node.js 20 или новее.
2. В папке проекта выполните `pnpm install`.
3. Запустите сайт командой `pnpm dev`.
4. Откройте `http://localhost:3000`.

## Что развивать дальше

- личный кабинет и сохранение прогресса;
- полноценный диагностический модуль из 1–3 кейсов с открытыми ответами;
- каталог уроков и база проверенных источников;
- редактор заданий, проекты и портфолио;
- кабинеты наставника, компании и университета;
- подписка, роли доступа и аналитика обучения.

Контент следует создавать на основе открытых материалов и академических источников, с указанием авторства и лицензий. Закрытые и нелицензированные материалы не копируются.
